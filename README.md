# Supported architectures:
- unix_amd64 (x86-64)
- mac_x86_64 (MacOS Intel)
- mac_aarch64 (MacOS Silicon)

# Updating WebP library
```
cd submodules/libwebp
git fetch
git checkout <tag>
```
